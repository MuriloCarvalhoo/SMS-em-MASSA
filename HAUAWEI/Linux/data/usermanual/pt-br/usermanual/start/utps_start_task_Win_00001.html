Iniciar o Mobile Partner

É necessário iniciar o Mobile Partner, antes de utilizar as várias funções do dispositivo do terminal.

Contexto

NOTA:
  • Quando o Número de Identificação Pessoal (PIN) for solicitado, insira o PIN correto. Se você não inserir o PIN correto ou o código da Chave de Desbloqueio do PIN (PUK), as funções relacionadas com a rede estarão indisponíveis.
  • Entre em contato com o seu fornecedor de serviços de rede para obter o cartão de Módulo de Identidade do Utilizador (UIM), o Módulo de Identidade do Subscritor (SIM) ou o Módulo de Identidade do Subscritor (USIM) .
  • Os cartões UIM, SIM ou USIM são fornecidos pelo seu fornecedor de serviços de rede. Entre em contato com o seu fornecedor de serviços de rede para mais informações.

Para iniciar o Mobile Partner manualmente

Clique duas vezes no ícone do atalho do Mobile Partner no ambiente de trabalho.

Para iniciar o Mobile Partner automaticamente

O Mobile Partner inicia automaticamente a partir da inicialização do sistema operativo.

Para definir que o Mobile Partner inicie automaticamente, faça o seguinte:

  1. Escolha Ferramentas > Opções > Geral.
  2. Selecione Executar na inicialização do OS a caixa de seleção.
  3. Clique em OK
