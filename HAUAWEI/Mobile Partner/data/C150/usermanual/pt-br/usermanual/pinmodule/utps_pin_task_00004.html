Verificação do PIN

Quando a verificação do PIN está disponibilizada, é necessário inserir o PIN para iniciar o Mobile Partner. As funções relacionadas com a rede (incluindo o acesso à Internet) do cartão UIM, SIM ou USIM só podem ser usadas após a introdução do PIN correto.

Contexto

NOTA:
  • Se um PIN incorreto for inserido, o sistema avisa sobre o tempo restante para que você insira o PIN correto.
  • Se você inserir o PIN errado consecutivamente além das vezes permitidas, o cartão UIM, SIM ou USIM é bloqueado e as operações relacionadas não podem ser realizadas.
  • É necessário introduzir o PUK para desbloquear o cartão UIM, SIM ou USIM.

Procedimento

  1. Escolha Ferramentas > Operações PIN > Inserir PIN.
  2. Introduza o PIN correto na janela exibida.
  3. Clique em OK para completar a verificação do PIN.
